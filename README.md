# 王府 店舗案内サイト

- トップページ: `index.html`
- 吉田町店ページ: `stores-yoshida.html`
- 吉田町店メニュー: `menus-yoshida.html`
- 藤枝店ページ: `stores-fujieda.html`
- 藤枝店メニュー: `menus-fujieda.html`
- 価格、営業時間、定休日、席数、駐車場、LINE URLは店舗確認後に更新
- 店舗情報は吉田町店と藤枝店で混在させない

## ローカル確認

```bash
node --check script.js
python3 -m http.server 8000
```
